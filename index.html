Installing Setup Timer
This application can be installed to your device by following these simple steps:
  1. Click on the share button at the bottom center of the browser.
  2. Scroll down and select "Add to Home Screen" from the menu that pops up.
  3. Select "Add" to install this application to the home screen.
The benefit of installing as an application is the ability to use it even when you do not have a network connection (for exmple, in some schools).

Installing Setup Timer
This application can be installed to your device by following these simple steps:
  1. Click on the three vertical dots in the upper right corner of the browser.
  2. Select "Add to Home screen" from the menu that pops up.
  3. Select "Install" to install this application (as opposed to adding a browser shortcut).
  4. Select "Install" to confirm that you'd like to install the application.
  5. Select "Add to home screen" to finalize installing the application to the home screen.
The benefit of installing as an application is the ability to use it even when you do not have a network connection (for exmple, in some schools).
